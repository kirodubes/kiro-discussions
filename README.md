The community space for **Kiro** ideas & suggestions. Suggest a feature,
upvote the ones you want, and help decide what gets built next.

- 👉 **Open the board:** https://github.com/kirodubes/kiro-discussions/discussions
- 🌐 **One-pager:** https://kirodubes.github.io/kiro-discussions/

> **This space is for ideas & suggestions — not bug reports.** Found a bug?
> Open it on the matching code repo. And every Linux distro is a good distro:
> we're here to make Kiro better, never to put down what you run.

## How it works

1. **Post it** — a feature request in **Ideas**, or a vote in **Polls**. A sentence is plenty.
2. **Vote it up** — upvote what you want. The board sorts by votes, so the most-wanted rise to the top.
3. **We build from the top** — the roadmap follows demand. A lot of the Arch Linux Tweak Tool and the ISO started as a request here.

## Categories

- 💡 **Ideas** — feature requests you can vote on
- 🗳️ **Polls** — we put a question to the community and you choose
- 🙌 **Show & tell** — share your Kiro setup, desktop, and tweaks
- 🙏 **Q&A** — ask the community for help

## Guidelines

- Be kind and respectful — to people and to other distros.
- Search before you post, then upvote instead of duplicating.
- Keep bug reports in the code repos; here we dream up features.
